صفحات الموقع
================================================================================

1. index.html — الصفحة الرئيسية
   • Hero سينمائي بصورة كاملة الشاشة
   • شريط إحصائيات متحرك
   • قسم نبذة عن الشركة
   • معاينة الخدمات (٦ خدمات)
   • قسم "لماذا نحن" (٦ بطاقات)
   • عرض المشاريع المميزة
   • خطوات منهجية العمل (٥ خطوات)
   • سلايدر آراء العملاء
   • شريط الشركاء المتحرك
   • بنر CTA بخلفية parallax
   • فوتر احترافي شامل

2. about.html — من نحن
   • قصة تأسيس الشركة
   • قسم الرؤية والرسالة
   • القيم الجوهرية (٤ قيم)
   • الإنجازات بالأرقام
   • Timeline تاريخي (٢٤ سنة)
   • بطاقات الفريق القيادي
   • أسباب ثقة العملاء

3. services.html — خدماتنا
   • ٨ خدمات كاملة مع تفاصيل وعلامات
   • بطاقة الخدمة المميزة (فلل فاخرة)
   • قسم منهجية العمل
   • إحصائيات الخدمات

4. projects.html — مشاريعنا
   • فلتر تصنيفات (الكل / فلل / تجارية / تشطيبات / ترميم / حديثة)
   • معرض Masonry بـ ١٢ مشروع
   • عرض مشروع مميز تفصيلي
   • إحصائيات المشاريع

5. contact.html — تواصل معنا
   • ٤ بطاقات معلومات الاتصال
   • ساعات العمل
   • نموذج تواصل كامل (٧ حقول)
   • Placeholder الخريطة
   • بطاقات الفروع الثلاثة
   • ٦ أسئلة شائعة Accordion

================================================================================
طريقة الاستخدام
================================================================================

1. فك ضغط الملفات في مجلد واحد
2. افتح index.html في أي متصفح حديث
3. لا توجد حاجة لخادم (Server) — يعمل مباشرة
4. للتخصيص: عدّل متغيرات CSS في :root بـ style.css

================================================================================
التخصيص السريع
================================================================================

تغيير اسم الشركة:
  ابحث عن "الفارس" واستبدله في جميع ملفات HTML

تغيير لون الذهبي:
  في style.css، غيّر قيمة --gold: #c8a96e إلى اللون المطلوب

تغيير الخط:
  في style.css، غيّر رابط Google Fonts ومتغيرات --font-display و --font-body

تغيير الصور:
  استبدل روابط Unsplash بصور مشاريعك الحقيقية
